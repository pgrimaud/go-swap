**Go-Swap** is a platform designed to enhance and streamline trading for players of the popular mobile game, Pok√©mon GO. Developed in PHP using the Symfony framework, it provides a smooth and user-friendly experience.

## Prerequisites

- PHP 8.2
- Symfony 6.3
- MySQL

## Installation

1. Clone this repository: `git clone https://github.com/pgrimaud/go-swap.git`
2. Install the dependencies: `composer install`
3. Start the Symfony server: Visit `http://localhost:8000` in your browser to access the application.
